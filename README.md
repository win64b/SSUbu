# SSUbu

Надежный Bash-скрипт для автоматизированной первоначальной настройки чистой ОС **Ubuntu Server 24.04 (LTS)**, развернутой в качестве виртуальной машины на **Hyper-V**.

## Особенности

- **Полное обновление системы**: Выполняет `apt update` и `apt full-upgrade -y`.
- **Интеграция с Hyper-V**:
  - Устанавливает необходимые пакеты интеграции (`linux-image-virtual`, `linux-tools-virtual`, `linux-cloud-tools-virtual`).
  - Принудительно устанавливает специфичные для текущего ядра утилиты: `linux-cloud-tools-$(uname -r)` для предотвращения ошибки `Unit not found`.
  - Включает и запускает службы обмена данными, теневого копирования и копирования файлов: `hv-kvp-daemon`, `hv-vss-daemon`, `hv-fcopy-daemon`.
- **Базовые утилиты**: Устанавливает набор инструментов администратора: `mc`, `htop`, `btop`, `curl`, `wget`, `net-tools`, `git`.
- **Удобство работы**:
  - Настраивает глобальный алиас `alias update='sudo apt update && sudo apt upgrade -y'` в `/etc/bash.bashrc` для всех пользователей.
  - Устанавливает часовой пояс `Europe/Moscow`.
- **Стабильность сервера**: Отключает (маскирует) спящие режимы (`sleep`, `suspend`, `hibernate`, `hybrid-sleep`).
- **Безопасность и контроль**:
  - Использование `set -e` для мгновенной остановки при любых ошибках.
  - Проверка запуска только с правами `root` или через `sudo`.
  - Проверка уже установленных пакетов для ускорения повторных запусков (идемпотентность).
  - Информативный цветной лог выполнения на каждом этапе.
  - Корректная обработка предложения о перезагрузке (интерактивный запрос или автоматический пропуск в CI/CD).

## Использование

1. Скачайте скрипт на ваш сервер Ubuntu 24.04:
   ```bash
   curl -O https://raw.githubusercontent.com/<ВАШ_ПОЛЬЗОВАТЕЛЬ>/SSUbu/main/setup_ubuntu_hyperv.sh
   ```
2. Сделайте его исполняемым:
   ```bash
   chmod +x setup_ubuntu_hyperv.sh
   ```
3. Запустите от имени root:
   ```bash
   sudo ./setup_ubuntu_hyperv.sh
   ```
